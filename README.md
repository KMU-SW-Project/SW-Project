# Bang Bang Pow
  * 이 게임은 VR 게임입니다.
  * 서부 시대를 배경으로 한 1:1 총싸움이 주 컨텐츠 입니다.
  * 현재 AI와의 결투만 구현 되어있습니다.

## 개발 도구
  * Unity 2019.4.91f(LTS)
  * SteamVR + VIVE VR

## 개발 기간
  * 2020.10 ~ 2020.12
  * 개발 완료후 일부 에셋 삭제
